Información sobre la API de Amiibos

Esta aplicación utiliza la API de Amiibos para obtener información sobre todas las figuras amiibo disponibles en el mercado. Gracias a esta API, podemos mostrar detalles como nombre, serie y características de cada amiibo.

Direcciones de la API utilizadas

  • https://www.amiiboapi.com/api/amiibo/ - Obtiene todos los amiibos
  • https://www.amiiboapi.com/api/amiibo/?name=##### - Obtiene información de un amiibo específico
